# ND-ZSH

A simple configuration using Nix flakes for ZSH. It is designed for minimal
environments

## Requirements

- Nix
- Nix flakes

## Installation

To quickly install and use such a shell run:

```bash
nix run github:NewDawn0/ND-ZSH
```
